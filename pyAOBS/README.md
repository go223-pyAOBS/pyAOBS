# 实时射线追踪功能

本项目已添加实时射线追踪功能，当用户修改速度模型时，系统会自动重新计算并更新射线路径和走时曲线。

## 功能特点

1. **模型修改自动触发更新**：当用户执行以下操作时，会自动触发射线追踪更新：
   - 移动模型节点
   - 插入新节点
   - 删除节点
   - 插入新图层
   - 删除图层
   - 修改速度值

2. **实时模式切换**：
   - 通过"模式"菜单中的"实时射线追踪"选项，可以打开或关闭实时射线追踪功能
   - 开启状态下，任何模型修改都会自动触发射线追踪更新

3. **射线显示**：
   - 显示详细的射线路径，包括射线序号、初始角度、最终角度等信息
   - 每条射线使用不同颜色，便于区分

## 使用方法

1. 打开速度模型（v.in）文件
2. 从"模式"菜单中启用"实时射线追踪"选项
3. 修改模型节点或速度值，系统将自动更新射线路径和走时曲线
4. 修改完成后，可以使用"射线追踪"按钮进行最终计算

## 技术实现

实时射线追踪功能通过在以下关键位置添加更新触发器实现：

1. 节点移动完成时（`on_button_release`）
2. 节点编辑函数（`update_node`）
3. 节点插入/删除操作（`insert_nodes`/`delete_nodes`）
4. 图层插入/删除操作（`insert_layer`/`delete_layer`）
5. 速度值编辑完成时（`edit_velocity_value`）

这些修改确保了在任何可能影响射线路径的操作后，都会自动触发射线追踪更新。
