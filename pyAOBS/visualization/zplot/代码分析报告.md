# ZPLOT 代码包详细分析报告

## 一、概述

ZPLOT 是一个用于读取地震剖面数据并进行交互式震相走时拾取的 C/Fortran 代码包。该软件包由 Colin A. Zelt 在剑桥大学 Bullard 实验室开发（1994年），后由 Haibo Huang 在 2023年进行了修改和扩展。

### 主要功能
1. **数据格式转换**：将 SEGY 格式地震数据转换为 Z 格式
2. **交互式显示**：图形化显示地震剖面
3. **震相拾取**：交互式拾取地震震相走时
4. **不确定性估计**：基于信噪比估计拾取不确定性
5. **数据输出**：将拾取结果转换为其他格式（如 rayinvr 输入格式）

## 三、核心模块详细分析

### 3.1 数据格式转换模块

#### 3.1.1 sgy2z/ - SEGY 到 Z 格式转换

**主要文件：**
- `sgy2z.f` - 主程序（Fortran）
- `fileio.c` - 文件 I/O 操作（C）
- `conv.c` / `conv.h` - 数据格式转换（IBM/IEEE 浮点数转换）
- `out_segy.c` - SEGY 输出处理
- `distance.f` / `geocar.f` - 距离和地理坐标计算

**功能：**
1. 读取标准 SEGY 格式地震数据
2. 支持多种数据格式：
   - `fmt='int'` - 2字节整数
   - `fmt='sun'` - 4字节 IEEE 浮点数
   - `fmt='ibm'` - 4字节 IBM 浮点数（转换为 IEEE）
3. 输出两个文件：
   - **数据文件 (dfile)**：Z 格式的地震数据
   - **头文件 (hfile)**：每条道的头信息（包含拾取信息）

**关键数据结构：**

```fortran
文件头 (fhead):
  - ntraces: 总道数
  - npts: 每道采样点数
  - sint: 采样间隔（微秒）
  - tstart/tend: 起始/结束时间（毫秒）
  - nrec: 记录数（炮集数）
  - npick: 拾取字数
  - vredf: 折合速度（km/s）

道头 (thead):
  - nrec: 炮站号
  - itsn: 道序号
  - irec: 接收站号
  - itype: 数据类型（1=垂直，2=径向，3=横向，4=水听器）
  - iflag: 数据标志（1=有效，其他=死道）
  - offset: 炮检距（米）
  - azi: 方位角（分）
  - igain: 增益因子
  - picks: 拾取时间数组（npick 个）
```

#### 3.1.2 su/ - Seismic Unix 格式处理

**主要文件：**
- `su2z.c` - SU 格式转 Z 格式（标准版）
- `su2z_hhb.c` - SU 格式转 Z 格式（修改版）
- `readsegy.c` - SEGY 读取工具
- `suxwigbx2.c` / `supswigbx2.c` - SU 格式数据可视化
- `surchwn.c` - SU 数据通道处理

**功能：**
- 将 Seismic Unix (SU) 格式数据转换为 Z 格式
- 支持 GEOMAR 扩展的 SEGY 头格式（240字节）
- 提供 SU 数据的可视化工具

**关键头文件：**
- `segyGEOMAR.h` - GEOMAR 扩展的 SEGY 头结构定义
- `segytape.h` - 标准 SEGY 头结构

### 3.2 主程序模块 - zplot/

#### 3.2.1 main.f - 交互式绘图和拾取主程序

**功能概述：**
这是整个软件包的核心程序，提供交互式的地震剖面显示和震相拾取功能。

**主要功能：**

1. **数据读取和显示**
   - 读取 Z 格式数据文件
   - 读取头文件（包含拾取信息）
   - 读取记录文件（rfile）- 包含炮集信息
   - 支持多种 X 轴显示方式（炮检距、道号、方位角等）

2. **交互式操作**
   - 键盘命令：
     - `1-5`: 改变 X 轴类型
     - `u/v`: 上下移动显示窗口
     - `-`: 改变 X 轴符号
     - `<,>`: 调整 X 轴范围
     - `0,6-9`: 改变折合速度
     - `c`: 切换裁剪模式
     - `p`: 拾取震相
     - `q`: 退出

3. **数据处理**
   - 带通滤波（bandpass）
   - 增益控制
   - 静校正
   - 水层校正
   - 折合时间显示

4. **拾取功能**
   - 鼠标点击拾取震相走时
   - 支持多个拾取字（npickw，最多40个）
   - 拾取结果保存到日志文件 `zplot.out`

**关键参数文件：**
- `zplot.par` - 参数定义文件
  ```fortran
  parameter (npmax=10000,    ! 每道最大采样点数
             ntmax=100000,   ! 最大道数
             nsmax=100,      ! 最大记录数
             npickw=40,      ! 拾取字数
             nptemp=1000)    ! 拾取模板点数
  ```

**输入文件：**
- `zplot.in` - 参数输入文件（namelist 格式）
- `dfile` - 数据文件（Z 格式）
- `hfile` - 头文件（二进制）
- `rfile` - 记录文件（ASCII，每行6列：记录号、X位置、Y位置、方位角、标题）

**输出文件：**
- `zplot.out` - 拾取日志文件
- `energy.out` - 道能量文件
- `z.out` - 道值文件

#### 3.2.2 readf.f - 数据读取子程序

**功能：**
- `openf()` - 打开数据文件，读取文件头
- `readhr()` - 读取头文件和记录文件
- `readtr()` - 读取单道数据
- `gettrc()` / `gettrci()` - 获取实型/整型道数据

**关键处理：**
- 折合时间计算
- 时间对齐（基于拾取）
- 数据格式转换（整数/浮点数）

#### 3.2.3 headup.f - 头文件更新程序

**功能：**
- 读取 `zplot.out` 日志文件中的拾取更新
- 更新头文件中的拾取值和死道标志
- 保存更新后的头文件

**使用流程：**
```bash
headup < headup.rsp
# headup.rsp 内容：
# test.hdr    # 头文件名
# 10          # 拾取字数
```

#### 3.2.4 z2tx.f - 拾取结果转换程序

**功能：**
- 将头文件中的拾取结果转换为 `tx.in` 格式（用于 rayinvr 射线追踪程序）
- 支持多炮集处理
- 支持拾取不确定性输入

**输入文件：** `z2tx.in` (namelist 格式)
```fortran
&par
  hfile='new.hdr'
  picku=0.05,9*0.10,    ! 拾取不确定性（秒）
  npick=10,
  xmod=44.952,48.750,... ! 模型位置（km）
  azimuth=5*90.,         ! 方位角
  nshot=5,               ! 炮集数
  tshift=...             ! 时间偏移（可选）
  xshift=...             ! 位置偏移（可选）
  amscal=...             ! 振幅缩放（可选）
&end
```

**输出文件：** `tx.in` - rayinvr 输入格式

### 3.3 拾取不确定性估计模块

#### 3.3.1 pickunc/ - 基于信噪比的拾取不确定性估计

**主要文件：**
- `main.f` - 主程序
- `bandpass.f` - 带通滤波
- `misc.f` - 辅助函数

**功能：**
- 分析每个拾取点附近的信噪比（SNR）
- 根据 SNR 估计拾取不确定性
- 使用经验关系：SNR 与拾取不确定性的对应关系

**输入文件：** `pickunc.in`
```fortran
&picpar
  dfile='data.z'
  hfile='header.hdr'
  ofile='pickunc.out'
  twin=0.25              ! 时间窗口（秒）
  pick=1,2,3,...         ! 要分析的拾取字
  snr1=...               ! SNR 值数组
  picku1=...             ! 对应的不确定性数组
  snr2=...
  picku2=...
  tlag=0.1               ! 时间滞后
&end
&fltpar
  ibndps=0               ! 是否带通滤波
  freqlo=3.              ! 低截止频率
  freqhi=15.             ! 高截止频率
  izerop=1               ! 零相位滤波
  npoles=8               ! 滤波器阶数
&end
```

**输出文件：**
- `pickunc.out` - 不确定性估计结果
- `pickunc.hdr` - 包含不确定性的头文件

#### 3.3.2 pickamp/ - 拾取振幅分析

**功能：**
- 分析每个拾取点的振幅
- 计算最大振幅、最小振幅、平均振幅
- 可用于振幅校正和质量控制

### 4.2 数据格式说明

#### Z 格式数据文件结构：

```
记录1（文件头）：
  52字节：文件头信息
  - ntraces (4字节整数)
  - npts (4字节整数)
  - sint (4字节整数，微秒)
  - tstart (4字节整数，毫秒)
  - tend (4字节整数，毫秒)
  - nrec (4字节整数)
  - npick (4字节整数)
  - vredf (4字节浮点数)
  - ifmt (4字节整数，1=浮点，0=整数)
  - 其他参数...

记录2-N（道数据）：
  每条道：
  - 道头：22+npick 个浮点数（88+4*npick 字节）
  - 数据：npts 个采样点
    * ifmt=1: 4字节浮点数
    * ifmt=0: 2字节整数
```

## 五、关键技术特点

### 5.1 数据格式转换

- **IBM/IEEE 浮点数转换**：`conv.c` 提供完整的转换函数库
- **字节序处理**：支持大端/小端字节序转换
- **多种数据格式支持**：整数、IEEE 浮点、IBM 浮点

### 5.2 交互式拾取

- **实时显示**：X11 图形界面，实时更新
- **多种显示模式**：正常模式、裁剪模式
- **灵活的时间轴**：支持折合时间、绝对时间
- **多拾取字支持**：每条道可拾取多个震相（最多40个）

### 5.3 数据处理

- **带通滤波**：Butterworth 滤波器
- **增益控制**：自动增益、手动增益
- **静校正**：支持时间校正
- **水层校正**：OBS 数据处理

### 5.4 不确定性估计

- **信噪比分析**：基于拾取点附近的时间窗口
- **经验关系**：SNR 与不确定性的对应关系
- **质量控制**：识别低质量拾取

## 六、编译和依赖

### 6.1 编译依赖

- **Fortran 编译器**：f77 或 gfortran
- **C 编译器**：cc 或 gcc
- **X11 库**：用于图形界面
- **Seismic Unix**（可选）：用于 SU 格式支持

### 6.2 编译命令

```bash
# 编译 zplot
cd zplot
make xzplot

# 编译 sgy2z
cd sgy2z
make sgy2z

# 编译 SU 工具
cd su
make
```

### 6.3 Makefile 结构

- 使用标准的 Fortran/C 编译规则
- 链接 X11 库：`-L/usr/X11R6/lib -lX11`
- 输出到 `../bin/` 目录

## 八、关键子程序详解

### 8.1 数据处理子程序（misc.f）

#### 8.1.1 remdc() - 去直流分量
```fortran
subroutine remdc(x,npts,n1,n2)
```
- **功能**：移除数组 x 的直流分量
- **方法**：计算 n1 到 n2 之间的平均值，然后从整个数组减去该值

#### 8.1.2 crscor() - 互相关分析
```fortran
subroutine crscor(pilot,ncrcor,x,npts,n0,nlag,lag,ratio)
```
- **功能**：使用互相关方法确定数组 x 相对于 pilot 道的最优时间滞后
- **应用**：用于道对齐和时间校正
- **特点**：使用希尔伯特变换增强相关性

#### 8.1.3 pick() - 自动拾取
```fortran
subroutine pick(seis,npts,nstart,nfini,tmin,time,nwind,dts,iflag,minenratio)
```
- **功能**：自动拾取震相到达时间
- **方法**：
  - 计算每个点两侧窗口的能量比
  - 选择能量比最大的点作为拾取点
  - 使用滑动窗口方法提高效率
- **参数**：
  - `nwind`: 窗口长度
  - `minenratio`: 最小能量比阈值

#### 8.1.4 vg2() - 变增益滤波
```fortran
subroutine vg2(x,npts,dts,tvg,pvg)
```
- **功能**：应用变增益（Variable Gain）滤波
- **原理**：将每个点除以以该点为中心的窗口内绝对值的 pvg 次幂之和
- **效果**：
  - `pvg < 0`: 增强高频
  - `pvg = 0`: 无效果
  - `pvg > 0`: 增强低频

#### 8.1.5 tcas() - 自适应叠加对齐
```fortran
subroutine tcas(nsi,pjgl,wb,dtcw,wl,dts,zv,ntpk,nptsk,ipltk,tshift,ratio)
```
- **功能**：非线性反演方案，用于确定最优道对齐
- **作者**：Brian L.N. Kennett (ANU, 2002)，后由 Haibo Huang 修改 (2023)
- **方法**：
  - 使用自适应叠加确定最优道对齐
  - 给定初始近似对齐（如模型预测的 AK135），可获得模型走时残差
  - 使用 Lp 范数进行叠加
- **参数**：
  - `nsi`: 叠加迭代次数
  - `pjgl`: 叠加指数（Lp 范数）
  - `wb`: 叠加窗口起始时间
  - `wl`: 叠加窗口长度
  - `dtcw`: 差分时间搜索范围
  - `tshift`: 输出时间偏移

#### 8.1.6 pstack() - 叠加处理
```fortran
subroutine pstack(pnkwb,pnkwl,pnsta,pdts,ratio)
```
- **功能**：对多道进行叠加处理
- **应用**：提高信噪比，增强弱信号

#### 8.1.7 smooth() - 平滑处理
```fortran
subroutine smooth(x,n)
```
- **功能**：对数组进行平滑处理
- **应用**：去除高频噪声

### 8.2 滤波子程序（bandpass.f）

#### 8.2.1 bndpas() - 带通滤波器设计
```fortran
subroutine bndpas(freqlc,freqhc,deltat,npole,izp,x,n,iflag)
```
- **功能**：设计和应用递归 Butterworth 带通滤波器
- **特点**：
  - 使用双线性 Z 变换设计滤波器，防止混叠
  - 截止频率处衰减 6dB
  - 滚降约 96dB/倍频程
  - 支持零相位滤波（`izp=1`）
- **参数**：
  - `freqlc, freqhc`: 低、高截止频率（Hz）
  - `deltat`: 采样间隔（秒）
  - `npole`: Butterworth 滤波器极点数
  - `izp`: 1=零相位滤波，0=非零相位

#### 8.2.2 filter() - 应用滤波器
```fortran
subroutine filter(x,n,ig,izp,npole)
```
- **功能**：将 bndpas 设计的滤波器应用到时间序列 x
- **方法**：递归滤波，可前向或前后向应用（零相位）

#### 8.2.3 hilbert() - 希尔伯特变换
```fortran
subroutine hilbert(x,n,ih)
```
- **功能**：计算希尔伯特变换
- **应用**：用于信号分析和互相关增强

### 8.3 显示子程序（shade.f）

#### 8.3.1 shade() - 阴影填充
```fortran
subroutine shade(sp,tp,np,ishade,ox,xscale,orig,tinch,dens,shadedc)
```
- **功能**：对道的正峰值（`ishade>0`）或负峰值（`ishade<0`）进行阴影填充
- **方法**：使用水平线填充
- **应用**：增强地震剖面的视觉效果

#### 8.3.2 intrtr() - 重采样
```fortran
subroutine intrtr(x,n1,n2)
```
- **功能**：通过线性插值将 n1 个点的数组重采样为 n2 个点
- **应用**：数据重采样和显示缩放

### 8.4 FFT 子程序（realft.f）

#### 8.4.1 realft() - 实数 FFT
```fortran
subroutine realft(data,n,isign)
```
- **功能**：对实数数组进行快速傅里叶变换
- **应用**：频域分析和滤波

#### 8.4.2 four1() - 复数 FFT
```fortran
subroutine four1(data,nn,isign)
```
- **功能**：对复数数组进行 FFT
- **应用**：通用频域处理

### 8.5 文件 I/O 子程序（fileio.c）

#### 8.5.1 open_input_() / open_output_()
- **功能**：打开输入/输出文件
- **特点**：提供 Fortran 可调用的 C 接口
- **文件句柄管理**：最多支持 21 个文件同时打开

#### 8.5.2 read_block_() / write_block_()
- **功能**：读写数据块
- **应用**：高效的数据传输

## 十、总结

ZPLOT 是一个功能完整的地震数据处理和震相拾取软件包，特别适用于：
- 宽角地震数据处理
- OBS（海底地震仪）数据处理
- 震相走时拾取和质量控制
- 地震层析成像数据准备

该软件包在地震学研究中广泛应用，特别是在海洋地震学领域。代码结构清晰，功能模块化，是一个成熟的地震数据处理工具。
