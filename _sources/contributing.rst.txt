Contributing
============

Issues and pull requests are welcome.

1. Prefer ``git add <specific-path>`` so unrelated local files are not committed.
2. Do not commit ``.venv``, large seismic test dumps, secrets, or tokens in remote URLs.
3. For GUI / petrology changes, note how you manually smoke-tested.

See also the root ``README.md`` contributing section and ``CHANGELOG.md``.
